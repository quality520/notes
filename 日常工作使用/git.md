git branch 查看分支
git branch -r 查看远程分支
git branch -a 查看所有分支
git branch -d 删除分支



git stash  隐藏修改   当pull的时候如果提示error可以使用此命令
git stash pop 显示隐藏    pull完成之后再将之前保存的恢复



git checkout  branchName   切换分支


git add -A
git commit -m "描述commit内容"


回到历史
git reset --hard commit_id  回到commit的那个版本
git reset --hard head
