#闭包
####闭包的概念
    -在计算机科学中，闭包（也称词法闭包或函数）是指一个
    函数或函数的引用，与一个引用环境绑定在一起。这个引
    用环境是一个存储该函数每个非局部变量（也叫自由变量）
    的表
    -闭包，不同于一般的函数，它允许一个函数在立即词法作用
    域外调用时，仍可访问非本地变量。
    优点：
      灵活和方便，封装
    缺点
      空间浪费
      内存泄漏
      性能消耗

#####全局、函数、eval
    javascript中没有块级作用域

    var a = 10;//全局作用域
    (function(){
      var b = 20;  //函数作用域
    })();
    console.log(a);//10
    console.log(b);//error,b is not defined

    for(var item in{a:1,b:2}){
      console.log(item);
    }
    console.log(item);//item still in scope
    eval("var a = 1");
